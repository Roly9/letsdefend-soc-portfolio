# SOC Analyst Portfolio – Let'sDefend Labs

This repository contains practical SOC analyst case studies based on real-world simulations completed on the Let'sDefend platform.

Each case demonstrates hands-on experience in:
- Threat detection
- Log analysis
- Incident investigation
- Attack classification
- Security decision-making
